# Stepan-2 — multi-tenant AI sales platform

Вторая версия Степана: **мультифилиальная** платформа AI-продаж. Каждый филиал —
изолированный арендатор со своей базой знаний, продуктами, персоной, языком, набором
пользователей и каналами (Instagram / Meta Business Suite / WhatsApp).

Строится **независимо от Stepan-1** (отдельный репозиторий, БД, контейнеры). Stepan-1
продолжает работать; миграция на Stepan-2 — по готовности (см. `docs/`).

## Принципы

- **Изоляция по `branch_id`** — данные филиалов не пересекаются (security-инвариант).
- **Дефолтного филиала нет** — у каждого филиала свой id.
- **LLM только через брокер** (AIbroker) — никаких локальных ключей провайдеров; учёт
  бюджета по `cost_usd` от брокера, бюджет на филиал.
- **Каналы за абстракцией** (`ChannelAdapter`) — чтение через MBS, фолоап через
  приватные API (Instagram, WhatsApp); единый чат лида не плодится между каналами.
- **SOLID / DRY**, полное покрытие тестами и актуальная документация.
